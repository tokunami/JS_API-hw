# JS_API-hw
13 JS API homework

### Usage
#### Development Environment
* python3
* pandas
* Flask
```
$ source activate {yourEnv}
$(yourEnv) pip install Flask
```
* sqlalchemy
```
$(yourEnv) pip install sqlalchemy
```

#### Run Flask App
```
$(yourEnv) cd JS_API-hw
$ python app.py
```
visit http://localhost/5000 on your browser
